### 2. Подключение
1. Скопируйте ключ доступа (начинается с `vless://`) из [Telegram-бота](https://t.me/outlivion_bot).
2. Откройте **V2Box**.
3. Приложение предложит вставить ключ из буфера обмена — подтвердите.
4. Если не предложило: перейдите во вкладку **Configs** -> нажмите **Import**.

<Callout type="info">
  При первом подключении macOS попросит разрешить добавление конфигурации VPN. Нажмите **Allow** и введите пароль от вашего Mac (или Touch ID).
</Callout>

## Вариант 2: Hiddify

Современный и красивый клиент.

### 1. Скачивание
[Скачать Hiddify для macOS (GitHub)](https://github.com/hiddify/hiddify-next/releases/latest)
*Для Mac на чипах M1/M2/M3 выбирайте файл с `arm64` в названии.*

### 2. Настройка
1. Установите `.dmg` файл.
2. Откройте приложение, нажмите **New Profile** -> **Add from Clipboard**.
3. Нажмите кнопку подключения.
